# Liuli Society — classic reference recreation

This is a GitHub-ready React + Vite website recreated from the classic reference style you shared.

## Important note
I could not verify the exact stock photos used in the screenshot, so this version uses the closest uncropped alternatives I found and downloaded from Unsplash and Pexels.

## Included pages
- Home
- About Us
- Pathways
- Research
- Community
- Support Us
- Contact

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
